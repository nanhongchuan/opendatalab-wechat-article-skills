**Vis3** **产品介绍**

  

![](https://mmbiz.qpic.cn/sz_mmbiz_png/hkUN8E95VAVZIVfibsRCmoL5MODxs0TV6AssP2GGgmeicIcbiafkJe5Gd8iaTmgaJNHl1CntU5E3plSuCRxwbMrXbA/640?wx_fmt=png&from=appmsg)

  

面向大模型领域的一站式数据可视化工具，专注于解决自然语言处理、机器学习等场景中复杂数据的解析与可视化难题。工具支持阿里云、AWS 等主流云端及本地数据的秒级接入，兼容 JSON、JSONL等多元格式。通过其智能识别数据结构、交互式可视化的能力，使数据以网页、Markdown、图片等视图模式清晰呈现，关键字段一目了然，大幅降低数据理解成本，助力用户快速洞察数据价值。

  

  

**Vis3** **特色功能**

  

01

**多格式数据支持**

无论是常见的 JSON、JSONL，还是特殊的 WARC 格式，Vis3 都能全面兼容。工具通过智能解析引擎自动识别数据结构，再配以精巧的可视化布局，将复杂数据中的关键信息以可视化的方式清晰呈现，显著节省用户在数据预览和理解过程中所耗费的时间与精力，让数据解读更高效。

### 

  

02

**交互式可视化**

点击即预览，Vis3 的交互式可视化功能带来便捷的操作体验。只需轻轻点击任意字段，即可实现该字段的单独预览，操作简单便捷。同时，它还提供了网页 / Markdown / 图片等多种灵活的视图模式，用户可根据自己的需求进行自由切换。无论是进行数据展示，还是深度分析挖掘，都能找到契合的呈现方式，让数据理解更透彻。

### 

  

03

**多源数据接入**

在数据接入上，Vis3 实现了云端数据与本地数据的无缝连接。阿里云、AWS、腾讯云等支持 S3 协议的主流云端存储平台，可轻松读取数据；本地数据文件也能秒级解析至可视化界面。灵活高效的数据接入方式，让你摆脱数据读取的繁琐和限制，专注于可视化分析。

  

  

**Vis3** **产品特性**

  

● 灵活：多源数据连接、多元格式兼容、多种视图自由切换，全方位满足不同场景的多元需求。

  

● 简易：点击即可预览数据，无复杂操作流程，大模型数据直观呈现，新手也能轻松上手。

  

  

**如何部署与使用**

  

Vis3提供了安装包，可本地部署使用。  

  

开始使用
----

    # python >= 3.9.2

或者使用 conda 创建 python 环境：（先自行安装conda）
------------------------------------

    # 1. 使用conda创建python3.11环境

  

---

更新到最新版本

    pip install vis3 -U

  

环境变量设置

ENABLE\_AUTH

开启用户登录鉴权

    ENABLE_AUTH=1 vis3

  

BASE\_DATA\_DIR

指定数据库文件（SQLite）目录

    BASE_DATA_DIR=your/database/path vis3

BASE\_URL

指定应用的资源 base 路径，影响 `/api` 路径。

    BASE_URL=/a/b/c

  

本地开发

    conda create -n vis3-dev python=3.11

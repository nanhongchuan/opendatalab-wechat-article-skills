****什么是Cherry Studio  
****

**功能强大的多模型 AI 客户端软件  
**  

  

Cherry Studio 是一款功能强大的多模型 AI 客户端软件，支持 Windows、macOS 和 Linux 等多平台运行，集成了 OpenAI、DeepSeek、Gemini、Anthropic 等主流 AI 云服务，同时支持本地模型运行，用户可以灵活切换不同的AI模型。

****如何在其启用MinerU  
****

**直接使用，也可私有化部署  
**  

  

得益于 Cherry Studio 与 MinerU 的官方合作，集成体验非常顺畅：

  

1.  1. 官方福利！免费额度畅享：  
    
2.  ● MinerU 官方每天为 Cherry Studio 每位用户提供了高达 500 页的文档预处理额度（限时两个月）。  
    
3.  ● 这意味着，在绝大多数情况下，您无需配置任何 API 密钥即可直接享受 MinerU 带来的强大功能！  
    
4.  ● 若您的日常使用量超出此额度，或本身有大规模文档处理需求，可以点击“获取密钥”按钮，前往 MinerU 官网申请您自己的 API 密钥。
    

![](https://mmbiz.qpic.cn/sz_mmbiz_png/hkUN8E95VAV3ojk5ZyrEnaWN2SN1WCrUgvpEXia28ZWBdnjVXHic1B9icLe0icTXnqGQhwDuYKyxpyGNRiaIIEWqD2w/640?wx_fmt=png&from=appmsg)

1.    
    
2.  2\. 灵活选择，满足多样需求：  
    
3.  ● 前往 Cherry Studio 的「设置」菜单。  
    
4.  ● 找到「工具」选项，点击进入「文档预处理」配置页面。  
    
5.  ● 在这里，您可以选择配置 MinerU 的 API 密钥（针对超额或高需求用户）。  
    
6.  ● 特别地，对于数据安全和隐私有极高要求的企业或个人用户，MinerU 的开源特性也使其支持私有化部署。 您可以将 [MinerU 服务完整部署在您自己的服务器上](https://mp.weixin.qq.com/s?__biz=MzkxMDc0MDU5Mg==&mid=2247550587&idx=1&sn=c5e384ed79be3ac63f6f8755b770fbfd&scene=21#wechat_redirect)，实现对数据的完全掌控，确保信息不出内网。部署完成后，您可以在此配置页面中填入您自部署 MinerU 服务的 API 密钥及地址，Cherry Studio 即可与之对接。
    
7.    
    
8.  3\. 在新建知识库时选择 MinerU：  
    
9.  ● 当您在 Cherry Studio 中新建知识库时，在「文档预处理」选项中，您会看到 MinerU 服务的选项。  
    
10.  ● 直接选择 MinerU，即可让您的新知识库在文档导入时，自动享受 MinerU 的专业级预处理服务。
     

![](https://mmbiz.qpic.cn/sz_mmbiz_png/hkUN8E95VAV3ojk5ZyrEnaWN2SN1WCrUvuOpwOGiaJHMoPuI9b6wlavYNhk7hb6EDkANB1pH0C5AGuKdJEWKJBg/640?wx_fmt=png&from=appmsg)

启用 MinerU 后，您上传到知识库的文档将经过更精细的处理，使 AI 对文档内容的理解更加深入和准确，从而显著提升知识库问答的质量和相关性。

****对话中的MinerU  
****

**更智能的交互（**功能预告**）  
**  

  

文档处理不仅限于知识库的构建，MinerU 的能力也完全融入了 Cherry Studio 的对话交互中。

  

当您与 AI 对话并引用复杂文档时，MinerU 将在幕后默默工作，帮助 AI 更透彻地理解文档中的图表、公式和深层含义，从而提供更精准、更富有洞察力的回答。这将显著提升 Cherry Studio 在处理复杂文档问答、信息提取等场景下的表现。

**MinerU 介绍及相关链接  
**

此次 MinerU 与 Cherry Studio 的官方合作，不仅仅是一次技术功能的集成，更助力 Cherry Studio 在文档处理智能化水平上迈上了一个决定性的新台阶。通过引入 MinerU 业界顶尖的预处理技术，并支持灵活的部署方式，能够极大地优化文档信息在 AI 应用中的流转效率和准确性，同时充分尊重和保障用户的数据主权，最终为每一位 Cherry Studio 用户带来更流畅、更智能、更值得信赖的产品体验。
